# OSRS-Auto-Cow-Killer
Use this completely modular image detection script to fight cows and collect their hides! 


## HOW TO USE:
- git clone repo
- cd into project directory
- pip install -r requirements.txt
- python3 cowkiller.py
- grab a coffee
